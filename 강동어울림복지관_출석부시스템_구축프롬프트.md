## 3. 실적 유형 정의

```
실(실인원)  : 중복 없이 집계한 참여자 수 (동일인이 여러 회기 참여해도 1명으로 카운트)
건(건수)    : 서비스 제공 횟수 (1:1 서비스, 방문, 상담 등 참여자 없이 집계 가능)
연(연인원)  : 회기별 참여자 수의 합산 (출석 체크 1회당 +1)
```

- **건수만 입력** 유형: 상담·홍보·행정·시설관리 등 참여자 명단 없이 건수로만 집계
- **실인원+연인원** 유형: 명단 기반 출석체크, 실인원·연인원 자동 산출
- **실인원+건수** 유형: 명단 기반이지만 건수를 병행 집계 (예: 사례관리, 1:1 서비스)

## 12. 핵심 비즈니스 로직

### 실인원 집계 알고리즘
```
특정 사업의 실인원 = 해당 사업 + 해당 기간(월 단위) 동안 출석O인 회원의 고유 수
(동일 회원이 같은 달에 10번 출석해도 실인원 = 1)
```

### 연인원 집계 알고리즘
```
연인원 = 해당 사업 + 해당 기간 동안 출석O 레코드의 총 합계
(동일 회원이 같은 달에 10번 출석하면 연인원 += 10)
```

### 건수 집계 알고리즘
```
건수유형 사업 : 직원이 입력한 건수 값의 합계
실인원유형 사업에서의 건수 : 별도 건수 필드 합계 (연인원과 무관)
```

### 달성률 계산
```
달성률(%) = (실적 / 목표) × 100
색상 표시 : 80% 미만 → 빨강, 80~99% → 노랑, 100% 이상 → 초록
```
